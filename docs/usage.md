# Usage

To use the Calculator, follow these steps:

1. Import the Calculator class:
   ```python
   from calculator import Calculator
   ```

2. Create a Calculator instance:
   ```python
   calc = Calculator()
   ```

3. Use the calculator methods:
   ```python
   result = calc.add(5, 3)
   print(result)  # Output: 8
   ```
